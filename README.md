# DebtDecoder

A personalized finance management app.

<details>

<summary>Built With</summary>

- Kotlin
- Firebase
- Android Studio
  
</details>

<details>

<summary>Modules</summary>

![image](https://github.com/user-attachments/assets/8bdd12af-eb15-4329-b710-c8571b9d490b)


- Login/Register/Forget Password
- Track Income
- Manage Expense
- Manage Debt
- Education Resources

</details>
